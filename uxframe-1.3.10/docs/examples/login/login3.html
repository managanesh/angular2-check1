Password Requirements

To ensure a safe and secure password, you must...

  • be 8 characters in length
  • include at least one digit and one alphabet character
  • start with an alphabet character
  • include only digits and lower case alphabet characters (1, 2, 3, a, b, c)
  • not have the same first three characters (eg: AAA123)
  • be used only one time and never be reused

Please close any open applications except for this browser window when changing your password.
