Libraries

As part of the UXFrame distribution, we have included a growing set of open source libraries that are frequently used in web application development. None of these are a necessary part of the core UXFrame functionality, but bring additional functionality that may be helpful in certain circumstances. We hope to expand the number of libraries included in the near future, so make sure to keep updated to the most recent version.

All of the Open Source Software (OSS) components included in UXFrame have been reviewed and approved for use by the Enterprise Open Source Team . Please contact them directly to address any additional questions regarding OSS licensing or distribution.

The lib/ directory contains minified versions of included libraries (including copyright information) as well as the related license files. To remain compliant with OSS license obligations, do not minify, obfuscate, or delete license text files included in our downloads.

JQuery Validate 1.14.0
License - Project

JQuery Validate is a basic form validation plugin. We provided the appropriate CSS on UXFrame form elements. Custom JavaScript is needed on a per-form basis to correctly validate user entry. Please note that this validation is purely for front-end experience and does not stand in for back-end validation. It would still be possible for a user to manually send a POST request with incorrectly formatted data. In this case, it would be expected that the back-end would reject the request and reprompt the user.
